¿Qué es CSRF y cómo puede prevenirse?

Cross Site Request Forgery, exploit malicisoso de un sitio web donde se envian peticiones o se realizan las mismas de manera no autorizada por un usuario logueado

La manera mas secilla de evitarlo es por medio de un token generado de manera aleatoria por el servidor, este se anexa a la forma cargada al usuario de manera oculta y a la vez se almacena en el servidor,al enviar la forma si el token coincide se registra la peticion
