Aparte de los arreglos $_POST y $_GET, ¿qué otros arreglos están predefinidos en php y cuál es su función?

  • $GLOBALS: Referencias a todas la variables que están definidas en el ámbito global del script

  • $_SERVER: Info de cabeceras, rutas y ubicaciones de script

  • $_FILES: Elementos subidos al script en curso a través del método POST

  • $_COOKIE: Variables pasadas al script actual a través de Cookies HTTP

  • $_SESSION: Variables de sesión disponibles para el script actual

  • $_REQUEST: Contiene el contenido de $_GET, $_POST y $_COOKIE

  • $_ENV: Variables pasadas al script actual a través del método del entorno

¿Qué es XSS y cómo se puede prevenir?

EL XXS se trata de un ataque de inyección de codigo malisioso para que se ejectue en sitios web, aplicaciones locales y en el navegador. Sucede cuando la persona no etica envia una pagina falsa con request falso para poder substraer datos de la persona destinada a ser atacada, esos datos que posteriormente son robados se mandan a una dirección de correo electronico o en mensajeria instantanea.

Se previene validando ambas partes de la pagina, tanto como del servidor y del usuario, haciendo uso de las funciones htmlspecialcharacters(),htmlentities() y deerminando cookies al igual que usando firewall.
