¿Qué ventajas y desventajas tiene la integración de tus aplicaciones web con servicios web desarrollados por terceros?

Se puede hacer uso de estas herramientas para facilitarse un poco de carga de trabajo al estar trabajando en proyectos grandes, es decir usar estas herramientas para no retrabajar casos de uso.

La desventaja es que al momento de implementar y tener errores o necesidad de mejor, se necesita la ayuda de 3eros.
