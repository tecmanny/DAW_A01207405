Question Answer
¿Cuáles son las diferencias entre los posibles valores de la propiedad position? Las propiedades de position son las siguientes:

Static: Valor por defectoque no define o define de manera estatica un elemento.
Relative: Hace lo mismo que Static a menos que le agregues otras propiedades.

Fixed: Posicionar un elemento del navegador de manera realtiva.

Absolute: Comportamiento similar al fixed pero relativo al la posicion anterior.
¿Cuáles son los valores estándar para la propiedad visibility? La propiedad visibility permite hacer invisibles o visibles las cajas de los elementos

Los valores permitidos son:
* Visible
* Hidden
* Collapse
* Inherit
¿Qué es el zIndex y para qué sirve? La propiedad Z-Index especifica el orden de apilamiento de un elemento
Un elemento con un valor de apilamiento mayor siempre va hasta el frente del que tiene menor valor.
