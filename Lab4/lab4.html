¿Qué diferencias y semejanzas hay entre Java y JavaScript?

- Java es un lenguaje orientado a objetos y Java Script es un lenguaje de scrpting orientado a objetos.
- Java crea aplicaciones que corren en una máquina virtual, mientras el código de JavaScript sólo corren en un navegador.
- El código de Java se debe de compliar, mientras que el de Java script está todo en texto.
- Requieren plug-ins diferentes.

¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)
 - getDate(): Obtiene la fecha como número
- getDay(): Obtiene el día de la semana como número (0-6)
- getFullYear(): Obtiene el año como numero de 4 dígitos (yyyy)
- getHours(): Obtiene la hora (0-23)
- getMilliseconds(): Obtiene los milisegundos (0-999)
- getMinutes(): Obtine los minutos(0-59)
- getMonth(): Obtiene el mes (0-11)
- getSeconds(): Obtiene los segundos (0-59)
- getTime(): Obtiene el tiempo (milisegundos desde 1 de enero de 1970)

¿Qué métodos tienen los arreglos? (Menciona al menos 5*)

- join(): Junta los elementos de un arreglo en un String.
- pop(): Quita el último elemnto de un arreglo y lo devuelve.
- push(): Agrega un nuevo elemento al final de un arreglo.
- shift(): Elimina el primer elemento de un arreglo y recorre hacia adelante los demás elementos.
- unshift(): Agrega un elemnto a la primera posición de un arreglo y recorre hacia atrás los demás elementos.
