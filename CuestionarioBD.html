Que es entonces una base de datos - DB?
Conjunto de información interrealacionada logicamente .

En que casos es conveniente usar bases de datos?
Se conviente usar una base de datos cuando se tienen diferentes areas de informacion en un corporativo o empresa relacionadas entre sí, como acerca de compras,ventes,producción.
Es recomendada para grandes escalas de información

Que es un sistema de gestion de base de datos - DBMS?
Una base de datos es una coleccion de datos que se encuentra interrelacionada y el conjunto de programas que se utiliza para acceder a estos datos.

Enlista y explica las funciones/responsabilidades que tienen los DBMS.

Implantacion de seguridad: Regular el acceso al contenido de los usuarios de las bases de datos.

Copias de seguridad y recuperacion: Detectar los fallos de la base de datos y restaurar la base de datos al estado en el que se encontraba antes del incidente.

Implantacion de la integridad: Los valores de los datos deben satisfacer las restricciones de consistencia.

Control de concurrencia: Controlar la interaccion de usuarios concurrentes y conservar la consistencia de los datos.
