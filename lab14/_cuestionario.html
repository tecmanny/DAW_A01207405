1. ¿Qué es ODBC y para qué es útil?
Las siglas significan Open DataBase Connectivity y es un estandar para conectarse a una base da datos, se hace uso de capas entre la aplicación y y el DBMS.



2. ¿Qué es SQL Injection?
Es un ataque para robar información o para modificar/destruir bases de datos que usan SQL.
